# Catálogo de Produtos

Aplicação desenvolvida em React que permite cadastrar, listar e excluir produtos
de forma dinâmica, utilizando estado e componentes.

## Funcionalidades
- Adicionar novos produtos
- Listar produtos cadastrados
- Excluir produtos da lista

## Tecnologias
- React
- JavaScript
- HTML5
- CSS3

## Como rodar o projeto
1. Clone o repositório
2. Instale as dependências:
   npm install
3. Rode o projeto:
   npm run dev
